📊 Excel  Dashboard Project

This repository contains an interactive Excel-based analytics dashboard designed to visualize key business performance metrics. The dashboard consolidates multiple data sources into easy-to-read visual reports for sales, profit, customer insights, and category-wise performance.

🔍 Key Features

• Sales by Category – Visual breakdown of sales contribution by product categories.
• Profit Gained – Profitability analysis with clear trend visuals.
• Monthly Sales Overview – Month-wise performance tracking.
• Sales by Representative – Sales distribution by salesperson.
• Top 5 Customers – Identification of the highest revenue-generating customers.
• Customer Count Insights – Overall customer engagement metrics.
• Interactive Dashboard – Clean consolidated view with slicers and KPIs.

🎯 Purpose

This project is ideal for:
• Business analysts
• Students practicing data visualization
• Teams needing a ready Excel dashboard template
• Portfolio demonstration of Excel BI skills
